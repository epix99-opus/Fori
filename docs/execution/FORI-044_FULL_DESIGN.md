### 9.3 OpenClaw 集成点（Wave 1 价格模块）

**SSE 端点**: `GET /api/v1/price/evaluate/stream`

**前端接入规格**（`/price/[communityId]` 页）：
```typescript
// 触发：点击「开始估价」按钮
const eventSource = new EventSource(`/api/v1/price/evaluate/stream?community_id=${communityId}&viewer_role=${role}`)

// 事件处理
eventSource.addEventListener('price:eval:start', () => setStatus('calculating'))
eventSource.addEventListener('price:eval:factor', (e) => appendFactor(JSON.parse(e.data)))
eventSource.addEventListener('price:eval:complete', (e) => setResult(JSON.parse(e.data)))
eventSource.addEventListener('price:eval:error', (e) => setError(JSON.parse(e.data)))
```

**UI 反馈规格**：
- `price:eval:start`：显示进度条（0%）+ 「AI 正在分析片区数据...」
- `price:eval:data_fetched`：「已获取 {sampleCount} 套成交样本」
- 每个 `price:eval:factor`：因子卡片逐一出现（动画展开）
- `price:eval:complete`：结果区块完整渲染 + 进度条消失
- `price:eval:error`：ErrorState 组件 + 重试按钮
